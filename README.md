# TravelWise

TravelWise is a beautifully designed, intuitive travel planning assistant built using Python and Streamlit. It helps users discover the perfect travel destinations across India by filtering dynamically based on budget, location, travel type, and ratings.

## Features

- **Interactive UI**: A highly polished, robust frontend to filter and explore destinations.
- **Smart Filtering**: Find the best trips by filtering through location (State/Region), expected max budget, travel style (e.g., Adventure, Heritage), and average ratings.
- **Destination Details**: Displays engaging cards with vital travel information, duration, accommodation, cost, and a quick Google Search integration for easy research.
- **Fast & Responsive**: Fully responsive UI taking advantage of Streamlit's caching and grid layouts.

## Tech Stack

- **Python**
- **Streamlit** (Frontend framework)
- **Pandas** (Data manipulation)

## Prerequisites

- Python 3.8+
- Active internet connection (for image fetching and Google search links)

## Running the App Locally

1. Clone the repository and navigate to the project directory.
2. Install the required dependencies:
   ```bash
   pip install streamlit pandas
   ```
3. Run the application from your terminal:
   ```bash
   cd .vscode
   streamlit run main.py
   ```
   *(Note: The main script is located in the `.vscode` directory based on the repository structure.)*
